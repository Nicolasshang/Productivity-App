# Reminders for git commands
1. Stage changes using `git add (file name)` or `git add .` to add everything.
2. Commit changes using `git commit -m "commit comment here."`
3. Push changes to the remote server using `git push origin (branch name)`. To push to the main branch, use `git push origin main`.
